Project Roadmap

Phase 1 — Research

Study Captain Claw file formats.

Tasks:

- Analyze REZ archives
- Identify asset structure
- Review existing tools such as NemoClaw

Phase 2 — Extraction Tool

Develop a tool capable of extracting assets from the REZ container.

Possible languages:

- Rust
- Python

Features:

- Parse REZ archive
- Extract files
- Maintain directory structure

Phase 3 — Integration

Prepare assets automatically for OpenClaw.

Tasks:

- Detect OpenClaw installation
- Copy extracted assets
- Validate installation structure

Phase 4 — Launcher

Create a launcher that starts OpenClaw with prepared assets.

Long Term

- GUI installer
- Mod manager
- Level editor
