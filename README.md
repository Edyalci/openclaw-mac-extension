OpenClaw Mac Extension

A toolkit to simplify running OpenClaw on macOS by automating asset extraction, setup, and launch of the game engine.

Overview

OpenClaw is an open-source reimplementation of the classic 1997 platform game Captain Claw.

While the engine itself runs on multiple platforms, users must manually extract assets from the original game installation. This process can be difficult, especially on macOS systems.

This project aims to create a toolkit that automates the asset extraction and configuration process.

Goals

The main goals of this project are:

- Provide an easy setup process for macOS users
- Extract game assets from the original REZ files
- Automatically configure OpenClaw
- Provide a simple launcher

Planned Features

- REZ asset extractor
- Automatic OpenClaw configuration
- CLI launcher
- macOS support
- Cross-platform compatibility (future)

Status
