# Contributing to OpenClaw Mac Extension

Thank you for your interest in contributing!

This project aims to make OpenClaw easier to run on macOS by providing tools, automation scripts, and improved installation workflows.

## Ways to Contribute

You can help by:

- Improving installation scripts
- Adding macOS compatibility tools
- Writing documentation
- Fixing bugs
- Creating automation tools

## Development Goals

The main goal is to simplify:

1. Installing OpenClaw on macOS
2. Extracting game assets
3. Running the engine smoothly on Apple Silicon

## How to Contribute

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a Pull Request

All contributions are welcome!
